Discuss the scope of var, let, and const?

The keywords var, let, and const are used to declare variables. The scope of a variable declared with var. This means that the variable can be accessed from anywhere. The scope of a variable declared with let is block scope. This means that the variable can only be accessed from within the block of code where it is declared. The scope of a variable declared with const is also block scope

Tell us the use cases of null and undefined

To represent the absence of a value. Null is used to represent the intentional absence of a value, while undefined is used to represent the unintentional absence of a value. To avoid errors. If you are not sure if a variable has been assigned a value, you can use null or undefined to avoid errors. In some cases, you may not know the value of a variable. In this case, you can use null or undefined to represent the unknown value.

What do you mean by REST API?

REST API stands for Representational State Transfer Application Programming Interface. It is an architectural style for designing APIs. REST APIs are based on the HTTP protocol and use standard HTTP methods to access and manipulate resources
